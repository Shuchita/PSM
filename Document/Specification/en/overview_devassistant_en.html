Overview

PlayStation(R)Mobile Development Assistant (PSM DevAssistant) is an application for developing a PSM application on an Android device or on PS Vita.

An Android version and PS Vita version of the application exist and the applicable version must be installed on the device prior to application execution.

Moreover, keys corresponding to the project must be created in advance on the relevant device using the Publishing Utility and a network connection enabled environment is required.

Regarding the setting on each device including installation and details of key creation, refer to the following documents.

  • Setup for Executing with an Android Device
  • Setup for Executing with PlayStation(R)Vita
  • Creating a Key to Execute a PSM Application on Actual Devices

Android Version PlayStation(R)Mobile Development Assistant

The following is the screen for the Android version PlayStation(R)Mobile Development Assistant.

Figure 1 Android Version PlayStation(R)Mobile Development Assistant

Select the menu at the upper right section described on Figure 1 to display the menu of the Android version PlayStation(R)Mobile Development Assistant.

Each menu item provides the following features.

Item name Description
Refresh Updates the list of installed PSM applications.
Debug Settings Displays various debug items.

The following items are under [Debug Settings].

Item name Description
Reset Ticket Information Clears purchase information regarding the In-App Purchase feature.
Sign In Inputs information for logging in to the PlayStation(R)Network.

Moreover, to delete an installed PSM application, select "Delete" from the menu displayed by long-pressing the cell of the applicable application name on the list.

PS Vita Version PlayStation(R)Mobile Development Assistant

The following is the screen for the PS Vita version PlayStation(R)Mobile Development Assistant.

Figure 2 PS Vita Version PlayStation(R)Mobile Development Assistant

Select the menu at the bottom right section of Figure 2 to display the option list of the PS Vita version PlayStation(R)Mobile Development Assistant.

Each menu item provides the following features.

Item name Description
Delete Contents Deletes an installed PSM application.
Intellectual Property Notices Displays trademark notices of PSM DevAssistant.
Go to the Developer Forums Starts a web browser and opens the Developer Forums page.
Debug Settings Displays various debug items.

The following items are under [Debug Settings].

Item name Description
Delete Contents Deletes save data (Documents) of all installed PSM applications.
Reset Ticket Information Clears purchase information of all installed PSM applications regarding the In-App Purchase feature.
Switch Development Assistant Switches the version of the running Development Assistant.

To create the Device Seed and keys using the PSM Publishing Utility and to install the PSM application, the version of the currently active Development Assistant must match the version of the PSM SDK Installer to use.

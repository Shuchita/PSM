About This Document

The PSM SDK (PlayStation(R)Mobile Software Development Kit) is a multi-platform application development and execution environment provided by Sony Computer Entertainment.

This document provides information for developing PSM applications using PSM SDK.

Developers using the PSM SDK for the first time to develop applications should refer to "About PlayStation(R)Mobile" and "Setup of the Development Environment".

Developers who are experienced in using the PSM SDK should check Release Notes for changes.

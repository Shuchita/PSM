UI 概要

Sce.PlayStation.HighLevel.UI は、UI を実現するための仕組みである UI Toolkit を提供します。UI Toolkit 上で動作する UI の部品やアニメーションも含まれます。

Sce.PlayStation.HighLevel.UI.dll をロードすることで使用可能になります。

ボタンやダイアログの表示が簡単に行え、また付属のレイアウトツールによりUI 部品の配置も簡単に行うことができます。

詳細は UI ToolKit プログラミングガイド および Sce.PlayStation.HighLevel.UI APIリファレンス を参照してください。

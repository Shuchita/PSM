Overview

The wireless controller driver for Windows is a driver for using the wireless controller of PS3(TM) in a Windows environment. By installing this driver, the wireless controller can be connected to a Windows PC and used as a USB game controller.

Note: Currently, only the PC simulator supports the wireless controller.

Installation

First, install the wireless controller driver as follows.

  1. Connect the wireless controller to the target PC with a USB cable.
  2. Open [System] from the [Control Panel], and open [Device Manager] from the [Hardware] tab.
  3. Open the properties of each [USB Human Interface Device] in order from the list of [Human Interface Devices]. Look for and select the device that includes the following string in its hardware ID that is displayed from the [Details] tab.
USB\VID_054C&PID_0268
  1. Move to the [Driver] tab of the selected device, and press the [Update Driver] button.
  2. A wizard will start up. Operate as follows.
  1. "No, not this time"
  2. "Install from a list or specific location (Advanced)"
  3. "Don't search. I will choose the driver to install."
  4. ("Show All Devices")
  1. The driver selection screen will be displayed. Click the [Have Disk] button and select the folder "%SCE_PSM_SDK%/tools/driver/sixaxis_driver/" storing the driver's sixaxis.inf file.
  2. [Wireless controller for PlayStation(R)3] will be displayed as the model name. Select it and proceed to the next step.
  3. A warning will be displayed indicating that the Windows logo test has not been passed. Ignore this warning and [Continue Anyway].
  4. This is the end of the installing process. Open [Game Controller] from the [Control Panel] and check that it is running properly.

Uninstallation

Uninstall the wireless controller driver as follows.

  1. Remove the wireless controller from the PC.
  2. Delete "C:WINDOWSsystem32driverssixaxis.sys" (the path will depend on the drive where Windows is installed).
  3. Delete the following registry keys using the registry editor, for example.
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USB\Vid_054c&Pid_0268
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\sixaxis

Troubleshooting

  • Installation does not complete correctly, and an error message such as "An error occurred while installing the device. The service installation section for this INF is invalid" appears.

-> Copy the driver file sixaxis.sys (for 32 bit or 64 bit version) to the following directory and then retry the installation.

C:/WINDOWS/system32/drivers (when Windows is installed to "C:/WINDOWS")

-> Unplug the SIXAXIS(TM) wireless controller from the PC, press the reset switch in the back, and then plug in the controller again. (The DUALSHOCK(R)3 wireless controller does not have a reset switch.)

  • The driver is installed, but input from the SIXAXIS(TM) wireless controller is not recognized.
  • On the device manager, the device is recognized but the "!" icon is displayed, indicating that it cannot be used.
-> Unplug the SIXAXIS(TM) wireless controller from the PC, press the reset switch in the back, and then plug in the controller again. (The DUALSHOCK(R)3 wireless controller does not have a reset switch.)

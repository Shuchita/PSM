usage:
        ModelConverter <input files> [options]
options:
        -o <filename>   specify output filename
        -s <scale>      scale geometry size
        -t <scale>      scale animation speed
        -S              output in text format
        -N              do not optimize
        -B              do not use BLENDINDICES
        -models         merge models into one file
        -motions        merge motions into one file

note:
    Autodesk(R) Crosswalk(R) 2013.0 is required in
    order to convert dotXSI files.

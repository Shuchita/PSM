このドキュメントについて

PSM SDK ( PlayStation(R)Mobile Software Development Kit ) は、ソニー・コンピュータエンタテインメントが提供する、マルチプラットフォームで動作可能なアプリケーションの開発・実行環境です。

このドキュメントは、PSM SDK を使ってPSMアプリケーションを開発するための情報を提供します。

初めてPSM SDKで開発を行う方は 「PlayStation(R)Mobileとは」「開発環境の設定」をご覧ください。

すでに開発をおこなったことのある方は リリースノート で変更点をご確認ください。

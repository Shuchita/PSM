Appendix

UI Composer で 利用することができるショートカットキーの一覧です。

表1 レイアウトウィンドウにおけるショートカットキー一覧

Function ShortCut Description
Create New Scene Ctrl+N 新たなシーンを作成します。
Close Current Scene Ctrl+W 開いているシーンを閉じます。
Create New Project Shift+Ctrl+N 新たなプロジェクトを作成します。
Open Existing Scene Ctrl+O 既存のシーンを開きます。
Save Ctrl+S 開いているシーンを保存します。
Build F5 コードの生成を行います。
Preview F12 シミュレータを起動し、シーンのプレビューを行います。
Undo Ctrl+Z 一つ前の状態に戻します。
Redo Ctrl+Shift+Z Undo の操作を一つ戻します。
Cut Ctrl+X Widget を Cut する
Copy Ctrl+C Widget を Copy する
Paste Ctrl+V Cut または Copy した Widget を 貼り付ける
Delete Del Widgetを削除する
Select All Ctrl+A Canvas 上の Widget をすべて選ぶ
Show Grid Ctrl+G Grid 線を表示する。
Arrow Cursor v Widget選択モードにする
Hand Cursor h スクリーン選択モードにする
Canvas Scale : 100% Ctrl+0 キャンバスの倍率を100%にします。
PlayStation(R)Mobile Documents F1 ドキュメントを開きます。

表2 オブジェクトツリービューにおけるショートカットキー一覧

Function ShortCut Description
Cut Ctrl+X Widget を Cut する
Copy Ctrl+C Widget を Copy する
Paste Ctrl+V Cut または Copy した Widget を貼り付ける

Preparation for Executing PSM Applications on Actual Devices

This document provides an explanation on the procedure for executing a PSM application on an actual device.

Procedure

1. Obtaining a publisher license

A publisher license is required to execute a PSM application on an actual device (Vita, Android device).

Obtain a publisher license from the SCE DevPortal site.

https://psm.playstation.net

Note: It may take a few days for a publisher license to be issued after it is requested.

2. Create keys

Keys must be created to execute a PSM application on a Vita or Android device.

Note: A "key" in this document refers to a Publisher Key, Device Seed, and application key.

Create keys as follows.

  1. Prepare PC for Internet connection.
Note: When using a proxy server with a company intranet environment, etc., refer to [Proxy Server Settings] below and make connection settings.
  1. Open the solution file of the PSM application to execute on the PSM Studio.
  2. Next, proceed to Setup for Executing with an Android Device or Setup for Executing with Vita.

Proxy Server Settings

When using a proxy server with a company intranet environment, etc. to connect to a network, make settings as follows.

  1. Double-click the [Publishing Utility] icon on the Desktop and start the Publishing Utility.

Figure 3 Publishing Utility

  1. Select [Menu] - [File] - [Proxy Server Settings]. A dialog will be displayed - make settings according to your environment.

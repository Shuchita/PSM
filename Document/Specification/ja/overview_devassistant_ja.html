PlayStation(R)Mobile Development Assistant について

この文書では実機でPSMアプリケーションを実行するために必要となる PlayStation(R)Mobile Development Assistant (PSM DevAssistant) について説明します。

Contents

  • 概要
    • Android版 PlayStation(R)Mobile Development Assistant
    • PS Vita版 PlayStation(R)Mobile Development Assistant

概要

PlayStation(R)Mobile Development Assistant (PSM DevAssistant) とは、AndroidデバイスやPS Vita上でPSMアプリの開発を行うためのアプリケーションです。

Android版 / PS Vita版 が存在し、アプリケーションの実行に先立って各デバイスにインストールされている必要があります。

また、各デバイス上での実行に先立って Publishing Utility を用いてプロジェクトに対応する鍵の作成とネットワークに接続できる環境が必要となります。

インストールを含む各デバイスでの設定、及び鍵作成の詳細についてはそれぞれ以下のドキュメントを参照してください。

  • Androidデバイスで実行するための設定
  • PlayStation(R)Vitaで実行するための設定
  • 実機でPSMアプリを実行するための鍵の作成について

Android版 PlayStation(R)Mobile Development Assistant

Android版 PlayStation(R)Mobile Development Assistant は、以下のような画面になります。

図1 Android版 PlayStation(R)Mobile Development Assistant

図1の右上のメニューを選択することで、Android版 PlayStation(R)Mobile Development Assistant のメニューが表示されます。

各メニューは以下のような機能を提供します。

項目名 概要
Refresh インストール済みのPSMアプリのリストを更新します。
Debug Settings 各種デバッグ項目を表示します。

[Debug Settings] 内の項目には、以下のようなものが存在します。

項目名 概要
Reset Ticket Information アプリ内課金機能について、購入情報をクリアします。
Sign In PlayStation(R)Networkにログインする為の情報を入力します。

また、インストール済みのPSMアプリを削除するには、リストに表示されている各アプリ名のセルを長押しすることで表示されるメニューから "Delete" を選択します。

PS Vita版 PlayStation(R)Mobile Development Assistant

PS Vita版 PlayStation(R)Mobile Development Assistant は、以下のような画面になります。

図2 PS Vita版 PlayStation(R)Mobile Development Assistant

図2の右下のメニューを選択することで、PS Vita版 PlayStation(R)Mobile Development Assistant のオプション一覧が表示されます。

各メニューは以下のような機能を提供します。

項目名 概要
Delete Contents インストール済みのPSMアプリを削除します。
Intellectual Property Notices PSM DevAssistant の権利表記を表示します。
Go to the Developer Forums ウェブブラウザーを起動して、Devloper Forums ページを開きます。
Debug Settings 各種デバッグ項目を表示します。

[Debug Settings] 内の項目には、以下のようなものが存在します。

項目名 概要
Delete Documents インストール済みの全てのPSMアプリのセーブデータ(Documents)を削除します。
Reset Ticket Information アプリ内課金機能について、インストール済みの全てのPSMアプリの購入情報をクリアします。
Switch Development Assistant 動作させる Development Assistant のバージョンを切り替えます。

PSM Publishing Utility によるデバイスシード・鍵の作成、及びPSMアプリをインストールするためには、現在アクティブな Development Assistant のバージョンと 使用する PSM SDK Installer のバージョンを合わせておく必要がありますので、注意が必要です。

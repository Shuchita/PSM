This document explains the procedure for executing a PSM application on PlayStation(R)Vita.

Execution on PlayStation(R)Vita requires a memory card and an environment with a wireless (WiFi/3G) network connection.

Preparation

Installing the USB Driver of PlayStation(R)Vita to the Development PC

If the OS of the development PC is Windows XP, you may be prompted to reinstall the USB driver upon using the USB cable to connect PlayStation(R)Vita and the development PC.

In this case, follow instructions in the Windows dialog and install the USB driver onto the development PC.

If the OS is Windows 7, there is no need to make this setting.

Installing the USB Driver to a Windows XP

  1. Connect the development PC with PlayStation(R)Vita using a USB cable.

When connected, the following dialog will be displayed; select "Yes, this time only" and click on the "Next >" button.

  1. Select "Install the software automatically (Recommended)" and click on the "Next >" button.
  1. Pressing the button will start the installing process.
  1. When installing of the USB driver completes, the following dialog is displayed.

Click on the "Finish" button to complete the installing process.

Installing PlayStation(R)Mobile Development Assistant to PlayStation(R)Vita

Next, install PlayStation(R)Mobile Development Assistant to PlayStation(R)Vita.

Use the following procedure to install Development Assistant.

  1. With the PlayStation(R)Network (Sony Entertainment Network) account registered in the PS Vita system, register as a developer on DevPortal.

  2. When developer registration is complete, Development Assistant can be downloaded from the PS Store. Start the PS Store application from the PS Vita Home screen.

  3. Tap the button in the lower right and select the Download List from the menu that appears.

  4. Select [Games] from the tab at the top.

  5. Search for [PlayStation(R)Mobile Development Assistant], then tap [Download] on the right to download and install.

  6. When installation is successful, the following icon will appear in the PS Vita Home screen.

Date and Time Settings

In the PS Vita Home screen, tap the [Settings] icon, then select [Date & Time] - [Date & Time Settings] - [Set via Internet] so that the date/time will be automatically obtained from the network. When the date and time have been obtained, synchronize the date/time of the PS Vita with the development PC.

Startup

Starting Up the PlayStation Mobile Development Assistant

  1. Connect the development PC to PlayStation(R)Vita using the provided USB cable with the memory card inserted to the PlayStation(R)Vita.
  2. Because it is necessary to connect to our server and go through authentication upon starting up a PSM application, make sure there is a wireless (WiFi/3G) connection established between PlayStation(R)Vita and the network.
  3. Tap the "PSM Dev" icon on the PlayStation(R)Vita home screen and start it up.

Note: When authentication succeeds on PlayStation(R)Vita, authentication will not be required again for the next 24 hours.

Settings on the PC

  1. Refer to Creating a Key to Execute a PSM Application on Actual Devices and create a key to be used for the project.

  2. Open any sample with PSM Studio and select PS Vita (xxxxxx) from the drop-down list.

    Note: (xxxxxx) indicates the ID number of PlayStation(R)Vita. The ID is unique for each PlayStation(R)Vita.

  3. From the menu, select [Run] - [Debug], or press the F5 key to transfer related files from the development PC to the memory card of PlayStation(R)Vita; the application will start up.

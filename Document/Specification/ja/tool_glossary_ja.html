用語集

用語 意味
レイアウトキャンバス (Layout Canvas) ウィジェットを配置するエリア。複数追加することができ、レイアウトウィンドウのタブで表示するレイアウトキャンバスを切り替えることができる。
ウィジェット (Widget) UI Toolkit で扱う UI 部品の単位。
レイアウトウィンドウ (Layout Window) 開発するユーザーインターフェースのレイアウトを作成するためのウィンドウ。レイアウトキャンバスが表示される。
アセットウィンドウ (Asset Window) 開発するユーザーインターフェースで使用するアセットフォルダー内を表示するウィンドウ。
オブジェクトツリー (Object Tree Window) レイアウトキャンバス上に配置されている UI 部品をツリー状に表示するウィンドウ。
ウィジェットリストウィンドウ (Widget List Window) UI Composer 内で配置できるウィジェットをリスト状に表示するウィンドウ。
オーバービューウィンドウ (Overview Window) レイアウトウィンドウに現在表示しているレイアウトキャンバスの全体像と表示領域を表示するウィンドウ。
プロパティウィンドウ (Property Window) 各ウィジェットで設定可能な項目を表示するウィンドウ。
言語テーブルウィンドウ (Language Table Window) 複数言語の文字列を ID 管理するためのウィンドウ。
ビルド (Build) 作成したレイアウトが .cs ファイルのコードとして出力される。
xxx.composer.cs ファイル レイアウト情報が記述された C# のソースコードファイル。
xxx.cs ファイル イベントハンドラなどを記述するための.C# のソースコードファイル。ユーザーが作成していなければ、ビルド時に自動で作成される。
xxx.uic ファイル UI Composer のプロジェクトファイル。プロジェクトの設定や、各レイアウト情報などが保存されている。
ガイド (Guide) ユーザーが設定するガイド線。ウィジェットを整列させる際に使用される。
スマートガイド (Smart Guide) ウィジェット間に表示されるガイド線。すでにレイアウトキャンバス内に配置されているウィジェットに対して位置やサイズを揃える際に有効な機能。
グリッド (Grid) レイアウトキャンバス上に表示される格子状の線。
